# IPEDTools

**IPEDTools** is a tool developed to complement the use of **[IPED (Indexador e Processador de Evidências Digitais / Indexer and Processor of Digital Evidence)](https://github.com/sepinf-inc/IPED)**, optimizing common tasks and adding useful features for digital forensic professionals.

**IPEDTools** é uma ferramenta desenvolvida para complementar o uso do **[IPED (Indexador e Processador de Evidências Digitais)](https://github.com/sepinf-inc/IPED)**, otimizando tarefas comuns e adicionando funcionalidades úteis para profissionais de análise forense digital.

## Features / Recursos

- **Process Automation / Automação de Processos**: Scripts and tools to streamline repetitive operations. / Scripts e ferramentas que agilizam operações repetitivas.
- **Report Handling / Manipulação de Relatórios**: Tools to convert and customize reports generated by IPED. / Ferramentas para converter e customizar relatórios gerados pelo IPED.
- **File Conversion / Conversão de Arquivos**: Support for converting data formats used in investigations. / Suporte para conversão de formatos de dados utilizados em investigações.

## Requirements / Requisitos

- **IPED**: Ensure IPED is installed and configured. / Certifique-se de que o IPED esteja instalado e configurado.
- **Java**: Version 8 or higher. / Versão 8 ou superior.
- Operating System / Sistema Operacional: Windows or Linux. IPEDTools has not been tested on macOS. / Windows ou Linux. O IPEDTools não foi testado em macOS.

## Installation / Instalação

1. Visit the [releases section](https://github.com/thiagofuer/IPEDTools_Releases/releases) of the repository. / Acesse a [seção de releases](https://github.com/thiagofuer/IPEDTools_Releases/releases) do repositório.
2. Download the file corresponding to your operating system: / Baixe o arquivo correspondente ao seu sistema operacional:
   - **IPEDTools.exe** for Windows. / **IPEDTools.exe** para Windows.
   - **ipedtools4linux.jar** for Linux. / **ipedtools4linux.jar** para Linux.

## Usage / Uso

1. Place the executable (**IPEDTools.exe** or **ipedtools4linux.jar**) in the IPED folder where the **iped.exe** or **iped.jar** file is located. / Coloque o executável (**IPEDTools.exe** ou **ipedtools4linux.jar**) na pasta do IPED onde está localizado o arquivo **iped.exe** ou **iped.jar**.
2. Run **ipedtools.exe** (on Windows) or **ipedtools4linux.jar** (on Linux) directly. / Execute o **ipedtools.exe** (no Windows) ou o **ipedtools4linux.jar** (no Linux) diretamente.
